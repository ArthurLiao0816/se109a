## 複雜度 BigO
* bubble sort : O(n^2)
* 2分搜尋 : O(log_n)
* 循序搜尋 : O(n)
* 合併排序 : O(n*log_n)
* 對於power2n.js的小想法~
    * `function power2(n) {
        if (n === 0) return 1
        return 2*power2(n-1)
        }`
    * 可以轉成...
    * `function power2(n) {
        let ans = 1
        for(let i = 0; i < n; i++){
        ans* = 2}
        return ans
        }`
    * 這樣可以更明顯的知道時間複雜度是BigO(n)
* 系統程式對於複雜度的觀點：不要超過O(n^2)會比較好
* 演算法對於複雜度的觀點：不要超過O(2^N)都好
